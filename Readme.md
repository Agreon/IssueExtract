# Issue-Extract

Extracts Issues from source-code and dynamically adds and deletes them from the remote github-repository

## Syntax
Git-Issue: <Title> >> <Body> <<

/** Git-Issue 'SomeTitle'
	'SomeText'
	['Label1','label2']
**/
